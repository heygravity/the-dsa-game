Lesson: Selection Sort

Selection Sort improves on Bubble Sort by reducing the number of swaps. It works by dividing the list into a sorted and an unsorted sub-list.

How It Works

  1. Find the smallest element in the unsorted portion.
  2. Swap that smallest element with the first element of the unsorted portion.
  3. Move the boundary of the sorted sub-list one element to the right.
  4. Repeat until the entire list is sorted.

The Logic in Code

for i from 0 to list.length-1:
  // Find index of minimum element
  minIndex = findMinIndex(list, from=i)
  // Swap it with the current element
  swap(list[i], list[minIndex])
